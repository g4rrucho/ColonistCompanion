Privacy Policy

Last Updated: August 15, 2025

Colonist Companion is a browser extension designed for Colonist.io players. It tracks in-game information such as opponent resources, development card usage, and dice statistics only during 1v1 games.

Information We Collect

This extension does not collect, store, or transmit any personally identifiable information, authentication credentials, financial data, health data, or location information.

All processing is done locally within your browser. Game-related data is read only from the Colonist.io game page you are currently playing on, and is not sent to any external server.

Permissions Justification

  • Host permission is required for https://colonist.io/* so the extension can read in-game messages and update its interface in real time.
  • Tabs permission is used only to detect if a Colonist.io game tab is active, so the extension can communicate with it.

Third-Party Code

Colonist Companion does not use remote code. All functionality is bundled into the extension package distributed via the Chrome Web Store.

Changes to This Policy

We may update this privacy policy to reflect changes in our extension or due to regulatory requirements. Any updates will be posted on this page with a revised "Last Updated" date.

For questions about this policy, please open an issue on our GitHub repository.
